Le musée des Augustins

La quasi-totalité des collections conservées au musée des Augustins est accessible par moteur de recherche à cette adresse. On peut ainsi accéder à l'inventaire des oeuvres acquises par le musée des Augustins depuis sa création en 1793. On trouvera sur ce site différents formats pour cet inventaire. On récupèrera notamment le fichier au format JSON . Une autre version de cet inventaire au format XML est également disponible.

  • Choisir quelques entrées communes à l'inventaire au format XML et au même inventaire au format JSON.

    • Faire une brève présentation de JSON.

      • JavaScript Object Notation (JSON) est un format de données textuelles dérivé de la notation des objets du langage JavaScript. Comme XML, il permet de représenter de l’information structurée. Un document JSON a pour fonction de représenter de l'information accompagnée d'étiquettes permettant d'en interpréter les divers éléments, sans aucune restriction sur le nombre de celles-ci. Un document JSON ne comprend que deux types d'éléments structurels : des ensembles de paires nom / valeur et des listes ordonnées de valeurs. Ces mêmes éléments représentent trois types de données : des objets, des tableaux ou des valeurs génériques de type tableau, objet, booléen, nombre, chaîne ou null. (source : Wikipédia)

    • Comparez les différentes entrées en parallèle.

      • Ressemblances

        Commme précisé dans la présentation de JSON, les formats JDSON et XML permettent de stocker de l'information structurée mais pas sous la même forme. Ainsi, on trouve dans le fichier XML l'équivalent de la paire clé/valeur "domaine":"Sculpture" en JSON sous la forme de l'ensemble balises/contenu <domaine>Sculpture</domaine>.

      • Différences

        Outre le nommage différent (par exemple mesure_3 en JSON et mesure3 en XML), certaines entrées ne stockent pas les mêmes informations. Ainsi, l'entrée "image" est plus développée en JSON qu'en XML ({"format": "JPEG", "thumbnail": true, "height": 1200, "width": 802, "filename": "066-119.jpg", "id": "68a3239dce120322b8794ac1cd6d867f"}).
