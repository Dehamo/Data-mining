Phase 2 : définir des requêtes xpath

  • Rechercher tous les NOM.

    • Requête : //taggertoken[contains(@pos,'NOM')]/@wordform

  • Rechercher tous les VERBES.

    • Requête : //taggertoken[contains(@pos,'VER')]/@wordform

  • Recherche tous les NOM précédés d'un DET.

    • Requête : //taggertoken[contains(@pos,'NOM')][preceding-sibling::*[1][contains(@pos,'DET')]]

  • Rechercher toutes les séquences NOM ADJ ou ADJ NOM.

    • Requête : //taggertoken[contains(@pos,'NOM')][following-sibling::*[1][contains(@pos,'ADJ')]] | //taggertoken[contains(@pos,'ADJ')][preceding-sibling::*[1][contains(@pos,'NOM')]]/@wordform
